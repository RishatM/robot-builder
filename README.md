# robot-builder

## Project setup

```
cd robot-api -> npm install -> npm start to run api server (robot-api serves images and data for client app)
```
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build

```

### Lints and fixes files
```
npm run lint

```
